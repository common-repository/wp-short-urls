== Description ==

This plugin allows the user to create short URLs from within your WordPress site. These URLs are easier for you to share and easier for your visitors to remember.  It creates a new menu item named Short URLs and allows you to add and delete short urls.

The origin would be from your WordPress site domain ( e.g. /download ) and the destination would be a full URL ( e.g.http://wordpress.org/download ).

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload entire wp-short-urls directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to the plugin page "Short URLs"
